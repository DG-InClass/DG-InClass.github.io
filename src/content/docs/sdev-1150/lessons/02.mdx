<Aside title="About the Demos...">
Many of the demo starter kits were initially set up by running `npm create vite@latest`. That *scaffolds* (sets up) a project that is not quite "bare-bones".

For a more minimalist setup, you can start with a truly empty web project and then manually add `vite` as the dev/web server and bundler.

<Steps>
1. Run these commands:

    ```ps title="Bare-Bones Setup"
    npm init -y
    npm install vite@latest
    ```

1. Edit `package.json`
1. Create an `index.html`
1. Run the web server `npm dev`
</Steps>
</Aside>

```js title="src/counter.js"
export function setupCounter(element) {
  let counter = 0
  const setCounter = (count) => {
    counter = count
    element.innerHTML = `count is ${counter}`
  }
  element.addEventListener('click', () => setCounter(counter + 1))
  setCounter(0)
}
```
